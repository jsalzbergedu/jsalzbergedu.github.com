Portfolio: Rust & Redox

My first contributions

Last year, in the summer of 2017, I got really into Rust and Redox. Rust is an exiting new systems-level programming language with a powerful type system and "borrow checker" that together help prevent bugs and use resources effectivley. Redox is a microkernel operating system written in rust. Over the summer, I worked on rust's orbfont, a high-level font loading and rendering font library for Redox as well as orbtk, Redox' widget toolkit.
